Какими бывают селекторы


Это выражения, которые говорят браузеру,
к какому элементу HTML нужно применить те или иные свойства CSS,
определённые внутри блока объявления стиля.

  • h1...це теговий селектор тут тілки р1 теги будуть вибраний
  • *...унівелсальний селектор
  • .h1...а це class селектор тут всо шо виділиний с classом буде ізменіний
  • #h1...id із цого селектора тільки один можебути
  • :...цей селектор виділяет активних силки
